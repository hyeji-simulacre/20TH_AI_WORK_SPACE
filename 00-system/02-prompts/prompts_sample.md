# Prompts 폴더 안내

## 이 폴더의 용도

자주 사용하는 프롬프트를 저장하고 재사용하는 공간입니다.

## 프롬프트 예시

### 요약 프롬프트

```
다음 내용을 3가지 핵심 포인트로 요약해주세요:
1. 주요 주장
2. 근거
3. 시사점

내용:
{붙여넣기}
```

### 번역 프롬프트

```
다음 영문을 자연스러운 한국어로 번역해주세요.
전문 용어는 원문을 병기해주세요.

원문:
{붙여넣기}
```

### 코드 리뷰 프롬프트

```
다음 코드를 리뷰해주세요:
- 버그 가능성
- 성능 개선점
- 가독성 개선점

코드:
{붙여넣기}
```

## 프롬프트 작성 팁

1. **구체적으로**: 원하는 출력 형식을 명시
2. **예시 포함**: 원하는 결과의 예시 제공
3. **역할 부여**: "당신은 ~전문가입니다"로 시작
4. **단계 구분**: 복잡한 작업은 단계별로 나누기

---
