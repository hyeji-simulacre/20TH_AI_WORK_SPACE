# 31. 노션 연동 (Notion Integration) - 나만의 디지털 지식 허브 구축

> **"로컬의 자료를 노션으로, 노션의 자료를 로컬로. 지식이 흐르는 파이프라인을 만듭니다."**

## 1. 🎯 개념 및 필요성 (Concept)

### 🧐 이게 뭐죠?
내 컴퓨터(로컬)와 클라우드 협업 도구(Notion)를 연결하는 기술입니다.<br> 파일을 자동으로 올리거나(`Upload`), 노션 페이지를 내 컴퓨터로 다운로드(`Download`) 할 수 있습니다.

### 💡 왜 배워야 하죠?
- **백업의 생활화**: 노션 서버가 터지거나 실수로 지워도, 내 컴퓨터에 백업본(`Markdown`)이 있으면 안심입니다.
- **자동 발행**: 내 컴퓨터에서 작성한 회의록을 AI에게 "노션으로 업로드해줘"라고 하면, 예쁘게 노션 페이지로 발행되어 팀원들에게 공유됩니다.
- **디지털 협업**: 내 컴퓨터의 지식 구조를 노션에서 팀원들과 편리하게 공유할 수 있습니다.

---

## 2. 🚦 단계별 접근 (Levels) - "왜 나눠져 있나요?"

문서 동기화(Sync)는 편리하지만, **"충돌(Conflict)"**이라는 무서운 문제가 있습니다. (내가 수정한 거랑 니가 수정한 거랑 섞이면? 💥)
그래서 우리는 안전하게 **단계별(Level)**로 접근합니다.

| 레벨 | 기능 | 방향 | 특징 | 
|---|---|---|---|
| **Lv1 (초급)** | **단방향 업로드** | 💻 로컬 → ☁️ 노션 | 팀 공유, 자료 발행 목적. (내 컴퓨터가 원본) |
| **Lv2 (중급)** | **분리된 양방향** | 💻 로컬 ↔️ ☁️ 노션 | 업로드와 다운로드 따로 실행. (버전 충돌 예방) |
| *Lv3 (고급)* | *전체 트리 동기화* | *(고급이상 세션)* | *폴더 구조 전체를 완벽하게 복제 (구조가 복잡해짐).<br> 데이터시스템 구축 수준으로 개발 고려해야 함.* |
| *Lv4 (전문)* | *실시간 동기화* | *(고급이상 세션)* | *드롭박스처럼 실시간 동기화 (충돌 위험 높음).<br> 데이터시스템 구축 수준으로 개발 고려해야 함.* |

> **우리는 가장 안전하고 실용적인 Lv1과 Lv2를 배웁니다.**

---

## 3. 🏭 지식 흐름 설계 (Workflow)

지식 관리 시스템에서 노션은 **"전시장"**입니다. 로컬 워크스페이스에 있는 자료 중 남들에게 보여줄 것만 골라서 전시장(`Notion`)으로 보냅니다.

- **Lv1 (Up)**: 로컬 폴더 → `Notion` (AI가 "어느 폴더의 파일을 업로드할까요?" 질문)
- **Lv2 (Down)**: `Notion` → 프로젝트 폴더 (AI가 "어느 프로젝트에 저장할까요?" 질문)

## 3. 🚀 Lv1: 업로드 (Upload) - "팀 공유하기"

### 1️⃣ 준비물
- 어떤 방식으로 파일을 노션으로 업로드할 수 있는지 AI에게 물어보세요(기능 이해하기).
- 업로드할 파일(`회의록.md`, `보고서.pdf`)을 준비하세요.
- AI에게 "어느 폴더의 파일을 업로드할까요?"라고 물으면, 폴더를 지정해주면 됩니다.

### 2️⃣ 명령어 입력

- "AI에게 무슨 ABC파일 노션으로 업로드해줘."라고 말하면 AI가 자동으로 실행합니다.
- "이 URL에 업로드 업로드 해줘."라고 말하면 AI가 알아서 실행합니다.

#### [Windows] PowerShell
```powershell
# 지정 폴더의 파일을 노션으로 업로드
python .claude\skills\notion-down-lv2\scripts\notion_upload.py --dir "20-created/21-ideas"

# [고급] 특정 파일을 특정 페이지로 업로드 (파일선택 + 페이지ID지정)
python .claude\skills\notion-down-lv2\scripts\notion_upload.py --dir "20-created/21-ideas" --files "회의록.md" --select-uploadpage-id "123456789abc..."
```

#### [macOS] Terminal
```bash
# 지정 폴더의 파일을 노션으로 업로드
python3 .claude/skills/notion-down-lv2/scripts/notion_upload.py --dir "20-created/21-ideas"

# [고급] 특정 파일을 특정 페이지로 업로드
python3 .claude/skills/notion-down-lv2/scripts/notion_upload.py --dir "20-created/21-ideas" --files "회의록.md" --select-uploadpage-id "123456789abc..."
```

### 3️⃣ 결과 확인
- 업로드가 성공하면 파일 이름이 **`(done)_회의록.md`**로 바뀝니다.
- 노션에 가보면 해당 페이지가 생성되어 있습니다!

## 4. 🚀 Lv2: 다운로드 (Download) - "백업하기"

### 1️⃣ 준비물 (Page ID)
- 어떤 방식으로 파일을 노션으로 다운로드할 수있는 AI에게 물어보세요(기능 이해하기).
- 다운로드하고 싶은 노션 페이지가 있어야 합니다.
- (환경 설정에서 다운로드할 기본 페이지를 미리 정해둡니다.)

### 2️⃣ 명령어 입력

- "AI에게 이 URL에 하위페이지를 다운로드해줘."
- "이 URL 단일페이지를 OO폴더로 다운로드 해줘."
 라고 말하면 AI가 알아서 실행합니다.

#### [Windows] PowerShell
```powershell
# 노션 페이지를 프로젝트 폴더로 다운로드
python .claude\skills\notion-down-lv2\scripts\notion_download_tree.py

# [고급] 특정 페이지를 특정 폴더로 다운로드 (페이지ID지정 + 저장폴더지정)
python .claude\skills\notion-down-lv2\scripts\notion_download_tree.py --select-downloadpage-id "123456789abc..." --output-dir "10-working/프로젝트명"
```

#### [macOS] Terminal
```bash
# 노션 페이지를 프로젝트 폴더로 다운로드
python3 .claude/skills/notion-down-lv2/scripts/notion_download_tree.py

# [고급] 특정 페이지를 특정 폴더로 다운로드
python3 .claude/skills/notion-down-lv2/scripts/notion_download_tree.py --select-downloadpage-id "123456789abc..." --output-dir "10-working/프로젝트명"
```

### 3️⃣ 결과 확인
- 실행 시 폴더 선택 메뉴가 표시됩니다. 프로젝트별 저장(`10-working/{프로젝트명}/`)이 권장됩니다.
- 선택한 폴더에 노션 페이지 내용이 `Markdown` 파일로 저장됩니다.

## 5. 💼 직무별 활용 사례 (Use Case)

### 🗓️ 프로젝트 매니저(PM) - "회의록 자동 배포"
- **상황**: 매주 회의록 작성 후 팀 노션에 올리고 링크 공유해야 함.
- **전**: 노션 켜고 -> 페이지 만들고 -> 내용 복붙하고 -> 제목 다듬고...
- **후**: VS Code에서 회의록 작성 -> AI에게 "노션으로 업로드해줘" -> **자동 업로드 완료** 및 팀공유. 

### ✍️ 콘텐츠 크리에이터 - "원소스 멀티유즈"
- **상황**: 블로그 글, 인스타 대본, 유튜브 기획안을 동시에 관리해야 함.
- **전**: 내 컴퓨터에서 작업하고 플랫폼마다 들어가서 따로 작성하고 저장 또 공유.
- **후**: 노션에서 팀원의 기획방향을 다운로드 -> 로컬에서 글 작성 -> 노션으로 업로드. 팀원 공유 후속업무 진행.
- -> 수정 사항은 다운로드해서 다시 백업.

### 📚 학습자/학생 - "나만의 위키 만들기"
- **상황**: 공부한 내용을 정리하고 싶은데, 노션은 예쁘지만 느리고 검색이 불편함.
- **전**: 노션에만 의존하다가 인터넷 안 되면 공부 못함.
- **후**: 노션 내용을 로컬로 다운로드 -> **Obsidian** 같은 도구로 연결해서 빠르게 정리. 오답노트까지 쉽게 정리하고 연결

### 👥 스터디장/커뮤니티 리더 D씨 - "자료 공유 자동화"
- **상황**: 매주 스터디원들에게 과제 가이드와 참고 자료를 배포해야 함.
- **후**: 내 컴퓨터에서 자료 정리 → AI에게 "노션으로 업로드해줘" → 스터디원들은 노션에서 보기. (일일이 카톡으로 파일 안 보내도 됨!)

## 6. 🍯 꿀팁: 업로드 했으면 '대시보드'를 꾸미세요!

우리가 파일을 올리면 노션의 '데이터베이스'에 쌓입니다.
1. 노션에서 **`Linked View of Database` (데이터베이스 링크된 뷰)**를 생성하세요.
2. 필터를 걸어서 **"최근 1주일간 올라온 문서"**만 보이게 하거나,
3. **"갤러리 뷰"**로 바꿔서 문서의 첫 이미지를 미리보기 하세요.

> **"AI로 노션 공유를 자동화한다면, 여러분은 노션의 페이지 구조만 체계적으로 잡으면 효율성이 몇 배 더 올라갈 것입니다.!"**


---

**💎 Tip1**: 처음에는 Lv1(업로드)부터 익숙해지세요. "내 컴퓨터에서 작성하고 노션으로 보낸다"는 흐름만 익혀도 업무 속도가 2배 빨라집니다. 그리고 구조를 잡으세요.
