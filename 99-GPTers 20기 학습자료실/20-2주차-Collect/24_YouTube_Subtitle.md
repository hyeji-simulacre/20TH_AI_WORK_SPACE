# 24. 유튜브 분석 - 영상에서 지식을 발굴하자!

> **"1시간짜리 영상을 언제 다 보고 있나요? 핵심만 3분 만에 읽으세요!"**

## 1. 🎯 개념 및 필요성 (Concept)

### 🧐 이게 뭐죠?
유튜브 영상이 어떤 내용으로 구성되어 있는지 컴퓨터가 읽을 수 있는 "메모장(JSON)" 파일로 가져오는 기술입니다.

### 💡 왜 배워야 하죠?
- **시간 절약**: 1시간짜리 강의 영상을 2배속으로 봐도 30분입니다. 분석한 내용을 읽으면 **3분**이면 됩니다.
- **AI 활용**: URL을 Claude에게 주면, "이 영상 요약해줘", "중요한 내용만 뽑아줘"가 가능해집니다.
- **데이터 소유**: 영상은 사라질 수 있지만, 저장해둔 분석한 레퍼런스와 나의 지식으로 새로운 지식을 만들 수 있습니다.

---

## 2. 🚀 이용 방법 (How-to)

### 1️⃣ 준비물 확인
- **파이썬**이 설치되어 있어야 합니다.
- **라이브러리/모듈**이 설치되어 있어야 합니다. 설치 가이드에 따라 AI가 자동으로 설치해줍니다. 

### 2️⃣ 명령어 입력 (복사 붙여넣기!)
- AI에게 `"URL"`만 전달하고 요구 사항을 말하세요.
- 그러면 AI가 자동으로 아래와 같이 실행합니다. 

#### [Windows] PowerShell
```powershell
# 예시: Steve Jobs 아이폰 발표 영상
python .claude\skills\youtube-content\scripts\yt-transcript-api.py "https://www.youtube.com/watch?v=MnrJzXM7a6o"
```

#### [macOS] Terminal
```bash
# 예시: Steve Jobs 아이폰 발표 영상
python3 .claude/skills/youtube-content/scripts/yt-transcript-api.py "https://www.youtube.com/watch?v=MnrJzXM7a6o"
```

### 3️⃣ 결과 확인
실행이 끝나면 탐색기(왼쪽 폴더 아이콘)에서 아래 경로를 확인해보세요.
- **경로**: `30-collected/32-youtube/`
- **파일**: `YYYYMMDD_영상ID_transcript.json`

---

## 3. 🤖 프로세스 이해하기 (Process)

1. **요청 (Input)**: 여러분이 영상 URL을 주면, 스킬(Python)이 유튜브 서버에 문을 두드립니니다.
2. **추출 (Process)**: "똑똑, 자막 있나요?" 물어보고, 한국어/영어 자막을 분석/처리합니다.
   *(자막이 없는 영상은 가져올 수 없습니다. ㅠㅠ)*
3. **저장 (Output)**: 깔끔하게 포장해서 `30-collected/32-youtube/` 폴더에 `JSON` 파일로 저장합니다.

## 4. 💼 직무별 활용 사례 (Use Case)

### 👩‍💼 마케터 A씨 - "경쟁사 분석의 신"
- **상황**: 경쟁사 신제품 발표 영상을 보고 보고서를 써야 함.
- **전**: 영상 3번 돌려보며 받아적느라 반나절 소요.
- **후**: 자막 추출 → Claude에게 "경쟁사 신제품 특징 3가지와 우리 제품과 비교해서 표로 만들어줘" → **10분 컷**.

### 👨‍🎓 대학생 B씨 - "A+ 학점 사냥꾼"
- **상황**: 교수님 온라인 강의 내용이 기억이 안 남.
- **전**: 강의 다시 듣기 (너무 지루함).
- **후**: 한 학기 강의 영상 전부 자막 추출 → 시험 기간에 "Ctrl+F"로 키워드 검색해서 공부.

### 🧑‍💻 개발자 C씨 - "기술 블로그 운영자"
- **상황**: 해외 개발 컨퍼런스(영어) 내용을 블로그에 정리하고 싶음.
- **전**: 영어 듣기 평가 하느라 멘붕.
- **후**: 영문 자막 추출 → Claude에게 "한글로 번역하고 기술 요약문 써줘" → **고퀄리티 포스팅 완료**.

### 🏢 1인 기업/프리랜서 D씨 - "트렌드 리포트 발행"
- **상황**: 매일 쏟아지는 업계 뉴스 유튜브 채널을 다 볼 시간이 없음.
- **후**: 주요 채널 5개 자막 추출 → "공통된 키워드 3개 뽑고, 이번 주 트렌드 요약해줘" → **주간 뉴스레터 발행**.

## 5. 🍯 꿀팁: Claude에게 뭐라고 질문하나요? (Prompt)

자막 파일(`json`)을 Claude에게 던져주고 이렇게 물어보세요.

> **"이 자막 파일 내용을 바탕으로 [초보자를 위한 3줄 요약]을 작성해줘."**
> **"여기서 언급된 [추천 툴 목록]만 따로 표로 정리해줘."**
> **"이 강의 내용을 [블로그 포스팅 말투]로 바꿔서 글을 써줘."**


---

**🎬 Tip**: 유튜브 자막 추출은 뒤에 배울 **"노션 연동"**과 합쳐지면 강력해집니다. 추출한 자막을 바로 노션에 저장할 수 있거든요!

## 6. ⚠ 한계점

유튜브에서는 자막 추출을 상당히 싫어합니다.

- 수시로 페이지 및 내부 로직을 변경합니다.
  - 이 경우 파이썬이 설치한 라이브러리가 개선 된 후 작동합니다.
- 시스템 방어 매커니즘이 강력합니다. 
  - 너무 자주 요청하거나 시스템에 부하가 있으면 차단되므로 적절한 시간 간격을 두세요.
- 다음장 개발 없는 대안 서비스(glasp)를 알고 있으면 문제가 발생하더라도 차선책으로 대응하기 좋습니다.(202)
