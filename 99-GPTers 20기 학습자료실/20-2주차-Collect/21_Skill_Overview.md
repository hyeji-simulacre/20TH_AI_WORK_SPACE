# 21. 지식 자동화 스킬 개요 (Skill Overview)

## 1. 🚀 '나만의 지식활용 AI워크스페이스'에 오신 것을 환영합니다!

우리는 **Claude Code**와 **자동화 스킬**을 활용하여 단순 반복 업무를 없애고, **진짜 중요한 일(기획, 분석, 창조)**에 집중하는 환경을 만듭니다.

### "Input → Process → Output" 자동화 공식
모든 지식 업무는 이 3단계로 이루어집니다. 우리가 배울 스킬들도 이 프레임워크 안에 있습니다.

| 단계 | 설명 | 사용할 스킬 (Skill) |
|---|---|---|
| **1. Input (수집)** | 외부 데이터를 내 컴퓨터로 가져오기 | • **유튜브 자막 추출** (영상 → 글)<br>• **PDF 리더** (잠긴 문서 → 글)<br>• **커스텀 스크래핑** (웹 → 글) |
| **2. Process (가공)** | 데이터를 정리하고 지식으로 만들기 | • **Claude Code** (AI 분석, 요약, 변환)<br>• **Python** (자동화 스크립트) |
| **3. Output (활용)** | 지식을 저장하고 공유하기 | • **Notion 연동** (지식 공유/발행)<br>• **Google Drive 연동** (자산 아카이빙) |

---

## 2. 📂 "도구는 공장에서 일합니다" (Johnny Decimal System)

우리는 스킬(도구)을 아무 데서나 쓰지 않습니다. **정해진 폴더(공장)**에서 실행해야 지식이 쌓입니다.

| 단계 | 폴더명 | 설명 | 비유 |
|---|---|---|---|
| **작업** | **`10-working/{프로젝트}`** | 현재 진행 중인 프로젝트 폴더 | **작업대** (지금 작업 중) |
| **생산** | **`20-created`** | 내가 만든 아이디어, 독서노트 | **내 머릿속** (내가 만든 것) |
| **수집** | **`30-collected`** | 외부에서 가져온 것 (웹, 유튜브, PDF) | **하역장** (외부에서 온 것) |
| **보관** | **`40-archive`** | 완료된 프로젝트 보관 | **창고** (끝난 것) |

> **💡 핵심 원칙**:
> 1. AI가 "어느 폴더에 저장할까요?"라고 질문하면, 용도에 맞는 폴더를 지정합니다.
> 2. **외부 수집물**은 `30-collected` 하위 폴더에 자동 저장됩니다.
> 3. **프로젝트 작업물**은 `10-working/{프로젝트명}/` 하위에서 관리합니다.
> 4. **프로젝트 완료 시** `40-archive`로 이동합니다.

## 3. 🎯 "내 업무는 이렇게 바뀝니다" (직무별 활용 사례)

### 👩‍💻 **마케터/콘텐츠 크리에이터**
> **"트렌드 조사하다가 하루가 다 가나요?"**
- **Before**: 매일 아침 뉴스레터, 유튜브, 경쟁사 블로그 수십 개를 직접 확인하고 스크랩하느라 오전 시간 소진.
- **After**: **[커스텀 스크래핑]** 스킬이 밤새 올라온 뉴스를 요약해서 **[Notion]**에 리포트로 만들어둡니다. "요즘 트렌드 뭐야?"라고 물으면 AI가 즉시 대답합니다.

### 📝 **기획자/PM**
> **"자료는 많은데 기획안 첫 줄 쓰기가 막막한가요?"**
- **Before**: 폴더 속에 흩어진 과거 산출물, PDF 보고서들을 찾느라 시간을 쓰고, 내용을 다시 읽어보느라 야근.
- **After**: **[PDF 리더]**가 수백 장의 보고서를 읽고 핵심만 추출해 저장합니다. "지난 프로젝트랑 비교해서 이번 기획안 초안 써줘"라고 하면 1분 만에 초안이 나옵니다.

### 💼 **지원/관리/운영**
> **"김대리, 그 파일 어디 있지? 질문받는 게 일인가요?"**
- **Before**: 팀원들의 반복되는 자료 요청에 파일 찾아서 메일 보내주고, 링크 다시 보내주느라 업무 흐름 끊김.
- **After**: **[Johnny Decimal]** 폴더 구조로 누구나 3초 안에 파일을 찾습니다. **[Google Drive 연동]**으로 팀원 모두가 실시간으로 최신 자료에 접근합니다.

## 5. 💡 자동화 스킬 사용 원칙

1. **"복잡한 코드는 AI에게 맡기세요"**
   - 여러분은 코드를 짤 필요가 없습니다. *"~해줘"*라고 명령만 하세요.
2. **"레고 블록처럼 조립하세요"**
   - 수집 스킬 + 노션 업로드 스킬 = **뉴스 자동 수집 및 스크랩 시스템**
   - 유튜브 스킬 + PDF 스킬 = **멀티미디어 학습 자료 생성기**
3. **"안전제일"**
   - 내 컴퓨터의 자료는 소중합니다. 모든 스킬은 안전하게 동작하도록 설계되었습니다.
