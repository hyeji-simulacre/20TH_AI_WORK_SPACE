## 페르소나 1: 김서연 (마케터)

| 항목 | 내용 |
|------|------|
| **나이/직업** | 29세, 스타트업 콘텐츠 마케터 |
| **경력** | 3년차, 1인 마케팅 담당 |
| **Pain Point** | "자료 조사에 하루의 절반을 쓰고 있어요" |
| **대표 니즈** | 자료 수집 |

### 상황

매주 블로그 2개, 뉴스레터 1개, SNS 콘텐츠 5개 제작. 트렌드 리서치에 2-3시간씩 소요. 크롬 탭 30개 열어놓고 작업하다 브라우저 크래시 경험 다수.

### 현재 행동

Notion에 링크 복붙 → 나중에 찾으면 "이게 뭐였지?" → 다시 검색

### 원하는 것

> "한 번 본 자료는 키워드로 바로 찾고 싶어요"

---

## 페르소나 2: 박준혁 (기획자)

| 항목 | 내용 |
|------|------|
| **나이/직업** | 35세, IT회사 서비스 기획자 |
| **경력** | 7년차, 팀 리드 |
| **Pain Point** | "분명히 정리했는데 필요할 때 못 찾아요" |
| **대표 니즈** | 지식 검색 |

### 상황

프로젝트마다 리서치 문서, 회의록, 기획서 생성. 폴더 구조가 프로젝트별로 달라서 과거 자료 찾기 어려움. "작년에 비슷한 거 했는데..." 하면서 결국 처음부터 다시 조사.

### 현재 행동

프로젝트 폴더 → 하위 폴더 5개 → 파일 20개 → Ctrl+F 반복

### 원하는 것

> "예전에 정리한 인사이트를 새 기획에 연결하고 싶어요"

## 페르소나 3: 이도윤 (AI 활용자)

| 항목 | 내용 |
|------|------|
| **나이/직업** | 32세, 프리랜서 콘텐츠 크리에이터 |
| **경력** | 유튜브 구독자 5만, 뉴스레터 운영 |
| **Pain Point** | "ChatGPT 대화창이 내 제2의 뇌인데, 휘발돼요" |
| **대표 니즈** | AI 통합 |

### 상황

GPT로 아이디어 브레인스토밍, 글 초안 작성, 자료 요약. 좋은 결과물이 나와도 대화창에 묻힘. 나중에 "그때 GPT가 뭐라고 했더라?" 하며 비슷한 프롬프트 다시 입력.

### 현재 행동

GPT 결과 → 복사 → 메모장에 붙여넣기 → 파일 어디 뒀는지 잊음

### 원하는 것

> "AI와 나눈 좋은 대화를 내 지식 체계에 쌓고 싶어요"
