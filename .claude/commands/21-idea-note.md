# Idea Note 생성

아이디어를 기록하는 노트를 생성합니다.

## 사용법

```
/21-idea-note [아이디어 제목 또는 내용]
```

### 예시

```
/21-idea-note AI로 회의록 자동 정리하는 서비스
/21-idea-note 팀 지식 공유 시스템 개선 방안
```

---

## 작업 흐름

### Step 1: 아이디어 파악

사용자가 제공한 아이디어 내용을 분석:
- 제목 추출 (20자 이내로 간결하게)
- 핵심 키워드 파악

### Step 2: 파일 생성

1. 파일명: `YYYY-MM-DD-제목.md`
2. 저장 경로: `20-created/21-ideas/`

### Step 3: Idea Note 작성

다음 구조로 생성:

```markdown
---
date: YYYY-MM-DD
type: idea
status: seed
tags:
  - idea
  - [관련 키워드]
---

# [아이디어 제목]

## 핵심 아이디어

[사용자가 제공한 아이디어를 1-2문장으로 정리]

## 배경/문제

- 왜 이 아이디어가 필요한가?
- 어떤 문제를 해결하는가?

## 구체화

### 어떻게 할 수 있을까?

-

### 필요한 것들

-

### 예상 결과

-

## 다음 단계

- [ ]

## 관련 메모

-

---

*작성일: YYYY-MM-DD*
```

### Step 4: 완료 안내

```
✅ Idea Note 생성 완료!

📄 파일: 20-created/21-ideas/YYYY-MM-DD-제목.md
💡 상태: seed (씨앗 단계)

아이디어를 더 발전시키고 싶으면 "구체화" 섹션을 채워보세요!
```

---

## 상태 표시

- `seed`: 씨앗 (막 떠오른 아이디어)
- `growing`: 성장 중 (구체화하는 중)
- `ready`: 준비됨 (실행 가능)
- `done`: 완료 (실행함)
- `archived`: 보관 (나중에 다시 볼 것)
