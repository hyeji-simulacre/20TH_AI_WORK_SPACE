# PKM Semantic Search Skill

**Gemini File Search**를 사용하여 내 노트에서 시맨틱 검색을 수행합니다.

---

## 트리거 키워드

다음 표현을 사용하면 자동으로 이 스킬이 실행됩니다:

- "~에 대해 검색해줘", "~찾아줘"
- "~뭐야?", "~뭐였지?"
- "~에 대해 알려줘"
- "전에 썼던 ~", "예전에 정리한 ~"
- "기억나?", "어디 적어뒀더라"
- "노트에서 ~", "내 문서에서 ~"

---

## 검색 명령어

```bash
# [TODO] 아래 경로를 자신의 볼트 경로로 수정하세요
cd "YOUR_VAULT_PATH/.pkm-search" && python3 -u query.py "검색어"
```

---

## 검색 결과 처리

검색 결과를 받으면:

1. **핵심 요약**: 가장 관련성 높은 정보를 bullet point로 정리
2. **출처 명시**: 어떤 문서에서 찾았는지 표시
3. **한국어 응답**: 결과를 자연스러운 한국어로 요약

---

## 사용 예시

### 예시 1: 아이디어 검색
**사용자**: "AI 활용 아이디어 정리한 거 찾아줘"
**검색어**: `AI 활용 아이디어`

### 예시 2: 프로젝트 정보
**사용자**: "마케팅 전략 회의록 뭐였지?"
**검색어**: `마케팅 전략 회의록`

### 예시 3: 학습 내용
**사용자**: "Claude Code에 대해 적어둔 거 있어?"
**검색어**: `Claude Code`

## 새 파일 동기화

PKM에 새 파일을 추가한 후:

```bash
cd "YOUR_VAULT_PATH/.pkm-search" && python3 -u sync.py
```
