# Archive 폴더 안내

## 이 폴더의 용도

완료된 프로젝트, 더 이상 활성화되지 않는 자료를 보관하는 공간입니다.

## 보관 기준

- **완료된 프로젝트**: `10-working/`에서 완료된 프로젝트
- **지난 기간의 노트**: 분기/연도별 정리 시 이동
- **참고용 자료**: 당장 필요하지 않지만 삭제하기 아까운 것들

## 폴더 구조 예시

```
40-archive/
├── 2024/
│   ├── Q4-프로젝트A/
│   └── Q4-프로젝트B/
├── 2025/
│   └── Q1-완료된프로젝트/
└── archive_sample.md
```

## 보관 시 체크리스트

- [ ] 프로젝트 상태를 `completed`로 변경했는가?
- [ ] 관련 링크가 깨지지 않도록 확인했는가?
- [ ] 필요한 경우 요약 노트를 작성했는가?
